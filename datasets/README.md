To use the provided annoation, download images and place them to the "images" forlder (for VOC it's the "data" folder), so the folders structure should be like:
```
<dataset root>
    |--train.json
    |--val.json
    |--images
        |--<image1>.png
        |--<image2>.png
        ....
```
